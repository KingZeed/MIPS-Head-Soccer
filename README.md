# MIPS-Head-Soccer

## Description

Headsoccer is simple 2D game that is played on a chipkit with I/O Sheild.

## Features

- Singleplayer mode that is a survival mode
- bots with easy and hard mode
- Leaderboard
- multiplayer mode
- Goal animation with leds
